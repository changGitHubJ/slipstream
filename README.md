Slipstream: Bicycle race with Q learning
====

Overview

- Make neural network learn bicycle race using Q Learning

## Requirement

- Ubuntu 18.04 LTS
- python 3.6.9
- tensorflow 1.14.0

## Usage

1. start train.py
1. start test.py to check results
